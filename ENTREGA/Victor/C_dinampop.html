Categoria: Análise de Dados

Design: Sim

Escrita: Não

Situação: Entregue

Descrição: A proposta do projeto era modelar um sistema de dinâmica populacional para raias, vieiras e tubarões. Além disso, foi necessário implementar algum fator exógeno ao sistema. O projeto foi individual e, como fator exágeno, foi escolhido o derramamento de petróleo.Esse tema foi escolhido devido os grandes derramementos que já ocorreram nos oceanos e que mataram milhares de animais marinhos. Essa tese foi confirmado através de meu modelo.

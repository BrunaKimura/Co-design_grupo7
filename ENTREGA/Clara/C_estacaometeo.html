Descrição: O projeto consistiu na criação de uma estação meteorógica com base em conceitos elêtrônicos. Esse projeto tinha como objetivo aplicar as teorias elétricas e de programação à um projeto de real aplicação profissional. Além disso, permitiu a prática do trabalho em grupo, aprimorando a liderança e a dinâmica de elaboração. Como obrigatoriedade, foi utilizado sensores de temperatura, pressão e humildade, além de um sensor extra de escolha do grupo, no caso o sensor de gás carbônico. A programação efetiva foi realizada através de um Arduino (placa eletrônica), cuja linguagem de programação é 'C'. Parte fundamental do projeto foram as calibrações dos sensores, pois era necessário que os sensores obtivessem uma precisão adequada a suas medidas. Essa etapa do projeto demandou muito tempo, pois os sensores possuem um tempo de resposta à estímulos naturais, portanto tivemos que aguardar cerca de 10 a 30 minutos para cada medição, sendo no total cerca de 50 medições.

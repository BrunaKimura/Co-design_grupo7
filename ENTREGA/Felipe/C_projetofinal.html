Descrição: O projeto envolveu o desenvolvimento de um aplicativo web ou um jogo que utilizasse uma biblioteca alternativa do Python. Nesse caso, foi utilizado Flask, HTML, CSS e Firecall para criar uma espécie de aplicativo de relacionamentos para cães chamado PETinder, onde o dono cadastrava seus cães e procurava um parceiro para eles. Além disso, o app também contava com páginas para doação e adoção de cães.

Descrição: O projeto teve como objetivo a análise de dados coletados sobre o comportamento das populações de raias, tubarões e vieiras tendo como base uma pergunta específica. Para chegar aos resultados conclusivos foi necessário passar pela etapas de elaboração de uma diagrama de estoques e fluxos, elaboração das equações que definem o sistema e validação do modelo.
